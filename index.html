JS数据类型转换

数据类型转换

JavaScript是一种动态类型语言，变量没有类型限制，可以随时赋予任意值。虽然变量没有类型，但是数据本身和各种运算符是有类型的。如果运算符发现，数据的类型与预期不符，就会自动转换类型。当然有些时候也会出现需要强制转换的时候。

强制转换

强制转换主要指使用字符串(String)、数值(Number)和布尔(Boolean)三个构造函数，手动将各种类型的值，转换成数字、字符串或者布尔值。

字符串(String())

可以讲任意类型的值转化成字符串。

String()可以将任意类型转换为字符串

//原始类型的转换
String(123) //"123"  数值：转为相应的字符串。
String("js") //"js"  字符串：转换后还是原来的值。
String(true) //"true"  布尔值：true转为"true"，false转为"false"。
String(undefined) //"undefined"  undefined：转为"undefined"。
String(null) //"null"  null：转为"null"。

引用值类型（对象、函数、数组）转换为字符串，解析步骤如下：

  1. 调用 toString()，如果返回原始值类型，则使用 String 构造函数处理该原始值；如果返回的是复合类型，则调用 valueOf()；
  2. 如果 valueOf() 返回原始值类型，则使用 String 构造函数处理该原始值；如果返回的是复合类型，则抛出相应的错误。
String({a: 1}) // "[object Object]"
String([1, 2, 3]) // "1,2,3"

注: 字符串与字符串相加 得到的是字符串拼接

字符串与任意类型相加 得到的也是字符串（拼接）

字符串与字符串 - * / % 得到的是NaN

数值(Number())

Number()可以将任意类型转换成数值

Number(324) // 324  数值：转换后还是原来的值
Number('\t\v\r324\n') // 324  字符串：如果可以被解析为数值，则转换为相应的数值
Number('abc') // NaN  字符串：如果不可以被解析为数值，返回NaN
Number('') // 0  空字符串转为0
Number(undefined) // NaN  undefined：转成 NaN
Number(null) // 0  null：转成0

parseInt()和parseFloat()

//从头开始解析，遇到非数字后立即停止返回，有基数（进制）转换，
//parseInt 输出整数
parseInt("123abc")  //123
parseInt("-123.456.789")  //-123
parseInt("abc")  //NaN

// 从头开始解析，遇到非数字后立即停止返回，浮点数没有基数（进制）转换
// parseFloat 可解析到第一个小数点
parseFloat("-123.456.789")  //-123.456
parseFloat("abc")  //NaN

布尔(boolean())

Number()可以将任意类型转换成布尔值

//原始类型转换为布尔值
Boolean(456)  //true
Boolean("456")  //true
Boolean(false)  //false
Boolean(undefined)  //false
Boolean(null)  //false

注:所有对象（包括空对象）的转换结果都是true，甚至连false对应的布尔对象new Boolean(false)也是true。

JS基本数据类型

前几天做了一个有关基础数据类型的测评，感觉炸了，虽然在用的时候，没什么问题，但面对测评时，还是细节决定成败啊，所以今天就打算写一个关于基础数据类型的博客，来温习一下

引言

在javascript中申明变量使用的关键字都是var，这点与其他的编程语言不尽相同，但是javascript亦含有五种基本的数据类型（也可以说是简单数据类型），它们分别是：Undefined，Null，Boolean，Number和String。还含有一种复杂数据类型—Object。这里我们不谈复杂数据类型

基础数据类型

5种数据类型：String、Number、Boolean、Undefined、Null

1.String      // 是字符串类型
2.Number      // 是数值类型
3.Boolean     // 是布尔类型
4.Undefined   // undefined(未知)类型
5.Null        // null (空)类型

1.字符串类型(String)

由单引号或双引号括起来的字符序列,任何字符串的长度都可以通过访问length属性获得

var str = "hello world"
var str2 = 'hello IIWangTao'

字符串中一些特殊的字面量，即转义序列

String()函数可以将任何类型的值转换为字符串，包括null转换为’null’、undefined转换为’undefined’。

2.数值类型(Number)

可正可负可小数

var num = 10;
var num2 = -10;
var num3 = 0.1;
// 默认十进制

NaN

  • 即非数值，其表示一个本来要返回数值的操作数，未返回数值的情况。比如任何数值除以0，本是不符合规范的，js里，这样的操作返回NaN（但是实际上，只有0除以0时返回NaN，其他则无穷值）。
  • 任何涉及NaN的操作都会返回NaN，NaN值与任何值都不相等，包括本身。

  • isNaN()函数，这个函数可以判断，传递的参数是否“不是数值”这里涉及数值转换的问题:

    isNaN(“10”) == false;

    isNaN(“blue”) == true;

Number()，parseInt()，parseFloat()。第一个可以用于任何数据类型，后两个，则专门用于把字符串转化为数值。

3.布尔类型(Boolean)

该类型有两个值；true、false

出现场景

  • 条件语句中自动执行相应的转换
  • 字面量或变量定义

注：true不一定等于1，false也不一定等于0

Bloolean()函数，可以将其他类型的值转换为布尔类型。

4.未知类型(undefined)

用var声明变量，但未对其加以初始化时，这个变量的类型就是undefined，且其默认初始化值为undefined。

var a;  // var a = undefined

尚未声明过的变量，使用typeof方法检测其数据类型

var a;
alert(typeof a); // 弹出 undefined

出现场景

  • 声明未赋值的变量
  • 获取对象不存在的属性
  • 无返回值的函数执行结果
  • 函数的参数没有传入

5.空类型(Null)

此类型只有null一个值，null值表示一个空对象指针

注：1.定义的变量准备在将来用于保存对象（即复杂的数据类型object），那么就该将该变量初始化为null。

var b = null;
alert(typeof b); // object

注：2.null是空对象指针，而[]是空数组，{}是空对象，三者不相同

注：3.null不能添加自定义属性

过渡

通过 CSS3的transition，我们可以在不使用 Flash 动画或 JavaScript 的情况下，当元素从一种样式变换为另一种样式时为元素添加效果。



CSS3 过渡是元素从一种样式逐渐改变为另一种的效果。



要实现这一点，必须规定两项内容：

  • 指定要添加效果的CSS属性

  • 指定效果的持续时间。
    注意： 如果未指定的期限，transition将没有任何效果，因为默认值是0。

    <!doctype html>
<html>
<head>
    <meta charset="utf-8"/>
    <title>过渡</title>
    <style type="text/css">
    div{
        width: 200px;
        height: 100px;
        background-color: red;

        /*property  
            表示哪个属性有过渡效果
            可以是  width
            height 这些属性
            也可以是all
        */
        /*transition-property: width;*/

        /*duration  
            过渡总时长  单是秒、毫秒
        */
        /*transition-duration: 3s;*/

        /*delay 
            几秒之后执行过渡动画
        */
        /*transition-delay: 1s;*/

        /*
            过渡的时候速度的一个曲线，
            可以设置先快后慢，逐渐加快、逐渐变慢，
            也可以通过贝塞尔设置自己的速度曲线
        */
        /*transition-timing-function: cubic-bezier(0.91, 0.18, 0.92, 0.21);*/

        /*复合写法*/
        transition: width 5s ease 0s;

        /*浏览器的适用过渡*/
        /*谷歌浏览器*/
        -webkit-transition:;
        /*火狐浏览器*/
        -moz-transition:;
        /*IE浏览器*/
        -ms-transition:;
        /*欧朋浏览器*/
        -o-transition:;
    }
    div:hover{
        width: 700px;
    }

</style>
</head>
<body>
    <div>鼠标放我身上  查看效果</div>
</body>
</html>

2D动画

主要使用下面五个方法：

  • translate()方法，根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。
  • rotate()方法，在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转。
  • scale()方法，该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数：
    scale（2,4）转变宽度为原来的大小的2倍，和其原始大小4倍的高度。
  • skew()方法，该元素会根据横向（X轴）和垂直（Y轴）线参数给定角度:
    skew(30deg,20deg)是绕X轴和Y轴周围20度30度的元素。
    <!doctype html>
    <html>
    <head>
        <meta charset="utf-8"/>
        <title>2D动画</title>
        <style type="text/css">
        div{
            width: 200px;
            height: 200px;
            background-color: red;

3D动画

3D平移分别有三个方向：

第一个方向：translateX(x) 定义 3D 转化，仅使用用于 X 轴的值，以X轴平移；
第二个方向：translateX(Y) 定义 3D 转化，仅使用用于 Y 轴的值，以Y轴平移；这个和以X轴方向差不多，也就是上下平移，如果平移的负值，则往Y轴的负方向，也就是往下平移，反之，向上平移
第三个方向：translateZ(z) 定义 3D 转化，仅使用用于 Z 轴的值，以Z轴平移；对于这个角度的值，就得有三维空间的想象力了，那么，往Z周的正方向来平移，也就是相当于与我们眼睛目视前方的距离越来越近
3D旋转:
rotateX(angle) 定义沿 X 轴的 3D 旋转。
rotateY(angle) 定义沿 Y 轴的 3D 旋转
rotateZ(angle) 定义沿 Z 轴的 3D 旋转

HTML5新标签

h5的新标签

H5增加了这些标签最大的作用就是语义化，便于读取和识辨。因此也更加适合做SEO，容易快速准确的被爬虫读取，就像head和 titile 这样的标签元素一样。同时一些移动设备在解析到这些标签的时候做出特殊提示或者更加人性化的展示

注: 1. h5 不兼容IE 6 7 8 ; 2. h5 支持自定标签（慎用！！）

新增的标签

1. <header>定义一个页面或一个区域的头部。写html文档的时候可以用于包裹整个网页的头部，里面也可以放导航栏 （新标签：nav标签),支持HTML5的全局属性和事件属性

<header class="top">
    <img src="" alt="">
    <p>我是p标签</p>
    <nav>
        <a href="作业.html">HONE</a>
        <a href="#work">WORK</a>
        <a href="#contact">CONTACT</a>
        <a href="作业1.html">JOIN US</a>
    </nav>
</header>

2. footer定义一个页面或一个区域的页脚。CSS网页的尾部，一般放一些版权、联系方式、地址（address标签）、等。支持HTML5的全局属性和事件属性。

<footer class="footer">
    <img src="" alt="" >
    <ul>
        <li>Infini Studio</li>
        <li>北京市朝阳北路107号院</li>
        <li>Email：infinstudio@foxmail.com</li>
        <li>QQ:7585917</li>
        <li>wei:@InfinStudio</li>
    </ul>
</footer>

3. <article>定义外部的内容。比如来自一个外部的新闻提供者的一篇新的文章、或者来自 blog 的文本、或用户评论、或者是来自论坛的文本。亦或是来自其他外部源内容。大分区，可以包裹多个<section>标签

<article class="footer1">
    <section>
        <p>我是p标签</p>
        <img src="">
    </section>
    <section>
        <p>我是p标签</p>
        <img src="">
    </section>        
</article>

4. <section>定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。支持HTML5的全局属性和事件属性。小分区，可以包裹某类的元素

<section>
    <p>我是p标签</p>
    <img src="">
</section>

5. <aside>定义两栏或多栏页面的侧边栏内容，如联系我们、客服、网站公告等内容。支持HTML5的全局属性和事件属性

<p>黄山旅游</p>
<aside>
        <h4>黄山介绍</h4>

6. <hgroup>用于对网页或区段（section）的标题进行组合。当我们的网页出现多个标题的时候，就可以使用hgroup把多个标题包裹在一起。支持HTML5的全局属性和事件属性。

<section>
     <hgroup>
        <h1>this is h1</h1>
        <h2>this is h2</h2>
    </hgroup>
    <p>this is p</p>
</section>

7. <details>用于描述文档或文档某个部分的细节。支持HTML5的全局属性和事件属性。

8. <summary>包含 <details> 元素的标题，<details>元素用于描述有关文档或文档片段的详细信息。<summary>元素应该是<details>元素的第一个子元素。

<details open>       <!-- open 控制默认的开关状态 -->
    <summary>1</summary>
    <summary>2</summary>
    <summary>3</summary>

</details>

<figcaption>定义<figure>元素的标题(caption)。<figcaption>元素应该被置于<figure>元素的第一个或最后一个子元素的位置

<figure>

新的<input>类型属性
autofocus 自动对焦，会自动选中当前的<input>，只有一个

说明

Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用。> 看到这里请不要被「标记」、「语言」所迷惑，Markdown 的语法十分简单。> 常用的标记符号也不超过十个，这种相对于更为复杂的 HTML 标记语言来说，Markdown 可谓是十分轻量的，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。

标题

#~####### 分别表示1至6级标题

引用

当你需要引入一段别人的话时，只需要在文本前加入>(大于号)即可。

图片与链接

因为两者之间只是一个!的区别，所以就放在一起说了

粗体与斜体

表格

代码框

如果在你编写时，需要显示代码，只要另起一行，按下tab键，然后再写就行了

分割线

这个很简单，只需要另起一行，连续输入三个***即可

列表

Git是分布式版本控制系统，GitHub可以托管各种Git版本库，并提供一个web界面。其目的是为了将文件上传到自己的github上
