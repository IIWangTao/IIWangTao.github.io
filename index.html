JS数据类型转换

数据类型转换

JavaScript是一种动态类型语言，变量没有类型限制，可以随时赋予任意值。虽然变量没有类型，但是数据本身和各种运算符是有类型的。如果运算符发现，数据的类型与预期不符，就会自动转换类型。当然有些时候也会出现需要强制转换的时候。

强制转换

强制转换主要指使用字符串(String)、数值(Number)和布尔(Boolean)三个构造函数，手动将各种类型的值，转换成数字、字符串或者布尔值。

字符串(String())

可以讲任意类型的值转化成字符串。

String()可以将任意类型转换为字符串

//原始类型的转换
String(123) //"123"  数值：转为相应的字符串。
String("js") //"js"  字符串：转换后还是原来的值。
String(true) //"true"  布尔值：true转为"true"，false转为"false"。
String(undefined) //"undefined"  undefined：转为"undefined"。
String(null) //"null"  null：转为"null"。

引用值类型（对象、函数、数组）转换为字符串，解析步骤如下：

  1. 调用 toString()，如果返回原始值类型，则使用 String 构造函数处理该原始值；如果返回的是复合类型，则调用 valueOf()；
  2. 如果 valueOf() 返回原始值类型，则使用 String 构造函数处理该原始值；如果返回的是复合类型，则抛出相应的错误。
String({a: 1}) // "[object Object]"
String([1, 2, 3]) // "1,2,3"

注: 字符串与字符串相加 得到的是字符串拼接

字符串与任意类型相加 得到的也是字符串（拼接）

字符串与字符串 - * / % 得到的是NaN

数值(Number())

Number()可以将任意类型转换成数值

Number(324) // 324  数值：转换后还是原来的值
Number('\t\v\r324\n') // 324  字符串：如果可以被解析为数值，则转换为相应的数值
Number('abc') // NaN  字符串：如果不可以被解析为数值，返回NaN
Number('') // 0  空字符串转为0
Number(undefined) // NaN  undefined：转成 NaN
Number(null) // 0  null：转成0

parseInt()和parseFloat()

//从头开始解析，遇到非数字后立即停止返回，有基数（进制）转换，
//parseInt 输出整数
parseInt("123abc")  //123
parseInt("-123.456.789")  //-123
parseInt("abc")  //NaN

// 从头开始解析，遇到非数字后立即停止返回，浮点数没有基数（进制）转换
// parseFloat 可解析到第一个小数点
parseFloat("-123.456.789")  //-123.456
parseFloat("abc")  //NaN

布尔(boolean())

Number()可以将任意类型转换成布尔值

//原始类型转换为布尔值
Boolean(456)  //true
Boolean("456")  //true
Boolean(false)  //false
Boolean(undefined)  //false
Boolean(null)  //false

注:所有对象（包括空对象）的转换结果都是true，甚至连false对应的布尔对象new Boolean(false)也是true。

JS基本数据类型

前几天做了一个有关基础数据类型的测评，感觉炸了，虽然在用的时候，没什么问题，但面对测评时，还是细节决定成败啊，所以今天就打算写一个关于基础数据类型的博客，来温习一下

引言

在javascript中申明变量使用的关键字都是var，这点与其他的编程语言不尽相同，但是javascript亦含有五种基本的数据类型（也可以说是简单数据类型），它们分别是：Undefined，Null，Boolean，Number和String。还含有一种复杂数据类型—Object。这里我们不谈复杂数据类型

基础数据类型

5种数据类型：String、Number、Boolean、Undefined、Null

1.String      // 是字符串类型
2.Number      // 是数值类型
3.Boolean     // 是布尔类型
4.Undefined   // undefined(未知)类型
5.Null        // null (空)类型

1.字符串类型(String)

由单引号或双引号括起来的字符序列,任何字符串的长度都可以通过访问length属性获得

var str = "hello world"
var str2 = 'hello IIWangTao'

字符串中一些特殊的字面量，即转义序列

String()函数可以将任何类型的值转换为字符串，包括null转换为’null’、undefined转换为’undefined’。

2.数值类型(Number)

可正可负可小数

var num = 10;
var num2 = -10;
var num3 = 0.1;
// 默认十进制

NaN

  • 即非数值，其表示一个本来要返回数值的操作数，未返回数值的情况。比如任何数值除以0，本是不符合规范的，js里，这样的操作返回NaN（但是实际上，只有0除以0时返回NaN，其他则无穷值）。
  • 任何涉及NaN的操作都会返回NaN，NaN值与任何值都不相等，包括本身。

  • isNaN()函数，这个函数可以判断，传递的参数是否“不是数值”这里涉及数值转换的问题:

    isNaN(“10”) == false;

    isNaN(“blue”) == true;

Number()，parseInt()，parseFloat()。第一个可以用于任何数据类型，后两个，则专门用于把字符串转化为数值。

3.布尔类型(Boolean)

该类型有两个值；true、false

出现场景

  • 条件语句中自动执行相应的转换
  • 字面量或变量定义

注：true不一定等于1，false也不一定等于0

Bloolean()函数，可以将其他类型的值转换为布尔类型。

4.未知类型(undefined)

用var声明变量，但未对其加以初始化时，这个变量的类型就是undefined，且其默认初始化值为undefined。

var a;  // var a = undefined

尚未声明过的变量，使用typeof方法检测其数据类型

var a;
alert(typeof a); // 弹出 undefined

出现场景

  • 声明未赋值的变量
  • 获取对象不存在的属性
  • 无返回值的函数执行结果
  • 函数的参数没有传入

5.空类型(Null)

此类型只有null一个值，null值表示一个空对象指针

注：1.定义的变量准备在将来用于保存对象（即复杂的数据类型object），那么就该将该变量初始化为null。

var b = null;
alert(typeof b); // object

注：2.null是空对象指针，而[]是空数组，{}是空对象，三者不相同

注：3.null不能添加自定义属性

多说评论

之前用hexo和github搭建了一个简单的个人博客，今天给大家分享的是如何给自己的博客添加评论功能，话不多说，现在就开始喽！！！

正文

多说

1. 先进入多说评论官网

2. 点击我要安装

3. 创建站点

注：

站点名称——github用户名；
站点地址——个人博客地址；
多说域名——随意(注：但要记住你输入的内容，在下面的操作会用到)；
其他的照着要求填

4. 创建完成后，会生成一段代码(通用代码)

复制

5. 进入当前配置文件，找到_config.xml文件，添加：

duoshuo_shortname：你的多说域名(只写你输入的内容)

6. 进入themes/landscape/layout/_partial/article.ejs(这里的 landscape 是默认的主题名，如果你换了主题，需要替换成自己的主题名)

把最后一段这个代码
<% if (!index && post.comments && config.disqus_shortname){ %>
    <section id="comments">
        <div id="disqus_thread">
            <noscript>Please enable JavaScript to view the <a href="//disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
        </div>
    </section>
<% } %>
替换成第4步的通用代码

7. 再将粘贴好的通用代码中的第一个div，替换成下面的div

<!-- 多说评论框 start -->
<div class="ds-thread" data-thread-key="<%= post.layout %>-<%= post.slug %>" data-title="<%= post.title %>" data-url="<%= page.permalink %>"></div>

完成上述操作，你就有了多说评论了

搭建个人博客

如何搭建个人博客

在使用hexo和github搭建个人博客的时候，虽然查了很多的文献资料，而且需要运行的指令也非常详细，但是依然有很多的坑，这里说一下我出现的问题多是can not get blog,如果大家也有出现这样类似的问题，可以尝试清除一下缓存…

好了，不多说了，大家动手来搭建属于自己的个人博客吧

准备工作

· 安装nodeJS

hexo是基于node.js的一种博客框架，这里我们可以去nodeJS官网去下载nodeJS，进入官网大家会看到两个版本，这里建议大家下载前者，因为相比较前者更稳定

· 安装git

Git是分布式版本控制系统，GitHub可以托管各种Git版本库，并提供一个web界面。其目的是为了将文件上传到自己的github上

· 申请github账号

一个属于自己的账号还是要有的，没有的可以点击下面链接去申请：

参考链接

搭建博客

使用npm工具安装hexo

$ sudo npm install -g hexo

测试hexo是否安装成功(可选)

$ hexo

初始化hexo文件夹

$ hexo init

git

$ npm install hexo-deployer-git --save

生成静态页面

$ hexo generate（hexo g 也可以）

启动本地服务

$ hexo server (hexo s 也可以)

此时终端会出现http://localhost:4000/，用浏览器将其打开，一个最初的用hexo搭建的个人博客就呈现在你的面前了哦！！！
