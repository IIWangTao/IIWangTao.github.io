Markdown

引言

前几天搭建了自己的个人博客，也给大家分享了如何添加多说评论，那我今天就给大家分享如何写博客，既然是写博客，那就需要一个编辑器，这里我用的是Markdown,大家可以点击，进官网下载。

说明

Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用。> 看到这里请不要被「标记」、「语言」所迷惑，Markdown 的语法十分简单。> 常用的标记符号也不超过十个，这种相对于更为复杂的 HTML 标记语言来说，Markdown 可谓是十分轻量的，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。

Markdown语法的简要规则

标题

#~####### 分别表示1至6级标题

注意：需要在#后加一个空格，才能编写，这是最标准的Markdown语法

引用

当你需要引入一段别人的话时，只需要在文本前加入>(大于号)即可。

图片与链接

因为两者之间只是一个!的区别，所以就放在一起说了

百度


注：这些符号记得要是英文状态下的

粗体与斜体

此语法很简单，只是多个*的区别

表格

表格是 Markdown 比较复杂的地方，冒号表明文字对齐方式

代码框

如果在你编写时，需要显示代码，只要另起一行，按下tab键，然后再写就行了

分割线

这个很简单，只需要另起一行，连续输入三个***即可

列表

列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格
