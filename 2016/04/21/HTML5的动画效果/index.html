倒影

特点：不占内容宽高；倒影的层级比文档流的高；倒影是相对标签进行倒影

<!doctype html>
<html>
    <head>
        <meta charset="utf-8"/>
        <title>倒影</title>
        <style type="text/css">
        img{
            display: block;
            margin:200px auto;
            /*倒影
                left/right/below/above
                 左    右     下    上
            */
            -webkit-box-reflect:below 5px
            -webkit-linear-gradient(
                rgba(255,255,255,0),
                rgba(255,255,255,0.5),
                rgba(255,255,255,1));
        }    
        h1{
            -webkit-box-reflect:below 5px    
            -webkit-linear-gradient(
                rgba(255,255,255,0),
                rgba(255,255,255,0.5),
                rgba(255,255,255,1));
        }
        </style>
    </head>
    <body>
        <img src="http://a1.qpic.cn/psb?/V11qI0BQ26Ocim/GEYixYXRzxQV0bcfmQqmVMD8Jk2MpevW2CxjMJiG1M8!/c/dAsBAAAAAAAA&bo=FwGmAAAAAAACB5M!&rf=viewer_4" alt="">
        <h1>我是h1</h1>
    </body>
</html>

渐变

为了创建一个线性渐变，你必须至少定义两种颜色结点。颜色结点即你想要呈现平稳过渡的颜色。
同时，你也可以设置一个起点和一个方向（或一个角度）。
这里只写了线性渐变和径向渐变的简单效果，深一些的希望大家去探索

<!doctype html>
<html>
    <head>
        <meta charset="utf-8"/>
        <title>渐变</title>
        <style type="text/css">
            div{
                width: 250px;
                height: 250px;
                border: 3px black solid;
                color: white;
                line-height: 250px;
                text-align: center;
                font-size: 25px;
                display: inline-block;
                /*border-radius: 50%;*/
            }
            /*线性（梯度）渐变*/
            div:nth-child(1){
                background:-webkit-linear-gradient(blue, red);
            }
            div:nth-child(2){
                background:-webkit-linear-gradient(left, blue, red);
            }
            div:nth-child(3){
                background:-webkit-linear-gradient(45deg, blue, red);
            }
            div:nth-child(4){
                background:-webkit-linear-gradient(top right, blue, red);
            }
            div:nth-child(5){
                background:-webkit-linear-gradient(top, blue 10px, red 170px);
            }
            div:nth-child(6){
                background:-webkit-linear-gradient(top, gold 10%,blue 30%, red 70%, green 90%);
            }
            div:nth-child(7){
                background:-webkit-repeating-linear-gradient(blue 10%, red 20%);
            }

            /*径向渐变*/
            .div1{
                background: -webkit-radial-gradient(blue, green)
            }
            .div2{
                background: -webkit-radial-gradient(top, blue, green)
            }
            .div3{
                display: block;
                width: 750px;
                background: -webkit-repeating-radial-gradient(blue 10%, green 20%)
            }
            .div4{
                width: 750px;
                background: -webkit-repeating-radial-gradient(circle, blue 10%, green 20%)
            }
            .div5{
                background: -webkit-radial-gradient(top, blue 10px, green 60px)
            }
            .div6{
                background: -webkit-radial-gradient(top, blue 10%, green 40%, gold 70%);
            }
            .div7{
                background: -webkit-repeating-radial-gradient(blue 10%, green 30%)
            }
        </style>
    </head>
    <body>
        <div>grad_1</div>
        <div>grad_2</div>
        <div>grad_3</div>
        <div>grad_4</div>
        <div>grad_5</div>
        <div>grad_6</div>
        <div>grad_7</div>
        <h1>上面的效果都是线性渐变</h1>
        <div class="div1">1</div>
        <div class="div2">2</div>
        <div class="div3">3</div>
        <div class="div4">4</div>
        <div class="div5">5</div>
        <div class="div6">6</div>
        <div class="div7">7</div>
    </body>
</html>

过渡

通过 CSS3的transition，我们可以在不使用 Flash 动画或 JavaScript 的情况下，当元素从一种样式变换为另一种样式时为元素添加效果。

CSS3 过渡是元素从一种样式逐渐改变为另一种的效果。

要实现这一点，必须规定两项内容：

  • 指定要添加效果的CSS属性

  • 指定效果的持续时间。
    注意： 如果未指定的期限，transition将没有任何效果，因为默认值是0。

    <!doctype html>

    <head>
    <meta charset="utf-8"/>
    <title>过渡</title>
    <style type="text/css">
        div{
            width: 200px;
            height: 100px;
            background-color: red;
            /*property  
                表示哪个属性有过渡效果
            可以是  width
            height 这些属性
            也可以是all
            */
            /*transition-property: width;*/

            /*duration  
                过渡总时长  单是秒、毫秒
            */
            /*transition-duration: 3s;*/
            /*delay 
            几秒之后执行过渡动画
            */
            /*transition-delay: 1s;*/
            /*
                过渡的时候速度的一个曲线，
                可以设置先快后慢，逐渐加快、逐渐变慢，
                也可以通过贝塞尔设置自己的速度曲线
            */
            /*transition-timing-function: cubic-bezier(0.91, 0.18, 0.92, 0.21);*/
            /*复合写法*/
            transition: width 5s ease 0s;
            /*浏览器的适用过渡*/
            /*谷歌浏览器*/
            -webkit-transition:;
            /*火狐浏览器*/
            -moz-transition:;
            /*IE浏览器*/
            -ms-transition:;
            /*欧朋浏览器*/
            -o-transition:;
        }
        div:hover{
                width: 700px;
        }
    </style>
</head>
<body>
    <div>鼠标放我身上  查看效果</div>
</body>

    2D动画

    主要使用下面五个方法：

  • translate()方法，根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。
  • rotate()方法，在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转。
  • scale()方法，该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数：
    scale（2,4）转变宽度为原来的大小的2倍，和其原始大小4倍的高度。
  • skew()方法，该元素会根据横向（X轴）和垂直（Y轴）线参数给定角度:
    skew(30deg,20deg)是绕X轴和Y轴周围20度30度的元素。
<!doctype html>
<html>
    <head>
        <meta charset="utf-8"/>
        <title>2D动画</title>
        <style type="text/css">
            div{
                width: 200px;
                height: 200px;
                background-color: red;
                line-height: 200px;
                color: white;
                font-size: 50px;
                text-align: center;
                transform-origin: 200px 200px;
            }
            .div1:hover{
                transform: rotate(360000deg) translate(1000px,400px) scale(0.5,0.1) skew(180deg) skewY(180deg);
                transition:all 100s ease 0.2s;
            }
            .div2:hover{
                transform: translate(400px,400px);
                transition:all 15s ease 0.2s;
            }
            .div3:hover{
                transform: scale(0.5,0.1);
                transition:all 15s ease 3s;
            }
            .div4:hover{
                transform: skew(180deg);
                transition:all 15s ease;
            }
            .div5:hover{
                transform: skewY(180deg);
                transition:all 15s ease;
            }
        </style>
    </head>
    <body>
        <div class="div1">旋转</div>
        <div class="div2">位移</div>
        <div class="div3">缩放</div>
        <div class="div4">形变</div>
        <div class="div5">Y形变</div>
    </body>
</html>

3D动画

3D平移分别有三个方向：

第一个方向：translateX(x) 定义 3D 转化，仅使用用于 X 轴的值，以X轴平移；
第二个方向：translateX(Y) 定义 3D 转化，仅使用用于 Y 轴的值，以Y轴平移；这个和以X轴方向差不多，也就是上下平移，如果平移的负值，则往Y轴的负方向，也就是往下平移，反之，向上平移
第三个方向：translateZ(z) 定义 3D 转化，仅使用用于 Z 轴的值，以Z轴平移；对于这个角度的值，就得有三维空间的想象力了，那么，往Z周的正方向来平移，也就是相当于与我们眼睛目视前方的距离越来越近
3D旋转:
rotateX(angle) 定义沿 X 轴的 3D 旋转。
rotateY(angle) 定义沿 Y 轴的 3D 旋转
rotateZ(angle) 定义沿 Z 轴的 3D 旋转

<!doctype html>
<html>
    <head>
        <meta charset="utf-8"/>
        <title>3D动画</title>
        <style>
            div{
                width: 200px;
                height: 200px;
                background-color: red;
                font-size: 30px;
                color: white;
                line-height: 200px;
                text-align: center;
                transition: all 2s ease 0.2s;
            }
            .div1{
                transform-origin: 0px 0px;
            }
            .div1:hover{
                transform: rotateX(360deg);
            }
            .div2:hover{
                transform: rotateY(360deg);
            }
            .div3:hover{
                transform: rotateZ(360deg);
            }
            .div4:hover{
                transform: translateZ(501px);
            }
            .div5:hover{
                transform: scale3d(2, 2, 2);
            }
            .eyes{
                perspective: 500px;
            }
            .box{
                transform-style: preserve-3d;
            }
        </style>
    </head>
    <body>
        <div class="div1">X旋转</div>
        <div class="div2">Y旋转</div>
        <div class="div3">Z旋转</div>
        <div class="div5">缩放</div>
        <div class="eyes">
            <div class="box">
                <div class="div4">Z位移</div>
            </div>
        </div>
    </body>
</html>
