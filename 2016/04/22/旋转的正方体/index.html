旋转的正方体

旋转的正方体

做了一个关于动画的小练习，挺好玩的，所以就分享给大家吧！！！代码很简单：

<!doctype html>
<html>
<head>
    <meta charset="utf-8"/>
    <title></title>
    <style type="text/css">
        .eyes{
            position: relative;
            margin: 200px auto;
        }
        .box{
            width: 300px;
            height: 300px;
            transform-style: preserve-3d;
            background-color: red;
            animation: move 5s infinite linear;
        }
        div{
            width: 300px;
            height: 300px;
            position: absolute;
            opacity: 0.7;
            line-height: 300px;
            text-align: center;
            font-size: 150px;
            color: white;
            transition: transform 2s; 
        }
        .div1{
            background-color: blue;
            transform-origin: 0px 0px;
            transform: rotateX(90deg) translateY(-150px); 
        }
        .box:hover .div1{
            background-color: blue;
            transform-origin: 150px 0px;
            transform: rotateX(90deg) translateY(-300px) translateZ(300px) scale(2,2); 
        }
        .div2{
            background-color: green;
            transform-origin: 300px 300px;
            transform: rotateX(-90deg) translateY(150px);
        }
        .box:hover .div2{
            background-color: green;
            transform-origin: 150px 300px;
            transform: rotateX(-90deg) translateY(300px) translateZ(300px) scale(2,2);
        }
        .div3{
            background-color: gold;
            transform-origin: 0px 0px;
            transform: rotateY(-90deg) translateX(-150px);
        }
        .box:hover .div3{
            background-color: gold;
            transform-origin: 0px 150px;
            transform: rotateY(-90deg) translateX(-300px) translateZ(300px) scale(2,2);
        }
        .div4{
            background-color: brown;
            transform-origin: 300px 300px;
            transform: rotateY(90deg) translateX(150px);
        }
        .box:hover .div4{
            background-color: brown;
            transform-origin: 300px 150px;
            transform: rotateY(90deg) translateX(300px) translateZ(300px) scale(2,2);
        }
        .div5{
            background-color: purple;
            transform: translateZ(-150px);
        }
        .box:hover .div5{
            background-color: purple;
            transform: translateZ(-150px) translateZ(-300px) scale(2,2);
        }
        .div6{
            background-color: orange;
            transform: translateZ(150px); 
        }
        .box:hover .div6{
            background-color: orange;
            transform: translateZ(-150px) translateZ(600px) scale(2,2);
        }
        @keyframes move{
            0%{transform: rotateY(0) rotateX(0);}
            100%{transform: rotateY(720deg) rotateX(720deg);}
        }
    </style>
</head>
<body>
    <div class="eyes">
        <div class="box">
            <div class="div1">1</div>
            <div class="div2">2</div>
            <div class="div3">3</div>
            <div class="div4">4</div>
            <div class="div5">5</div>
            <div class="div6">6</div>
        </div>
    </div>
</body>
</html>
