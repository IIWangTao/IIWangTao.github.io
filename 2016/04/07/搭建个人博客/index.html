搭建个人博客

如何搭建个人博客

在使用hexo和github搭建个人博客的时候，虽然查了很多的文献资料，而且需要运行的指令也非常详细，但是依然有很多的坑，这里说一下我出现的问题多是can not get blog,如果大家也有出现这样类似的问题，可以尝试清除一下缓存…

好了，不多说了，大家动手来搭建属于自己的个人博客吧

准备工作

· 安装nodeJS

hexo是基于node.js的一种博客框架，这里我们可以去nodeJS官网去下载nodeJS，进入官网大家会看到两个版本，这里建议大家下载前者，因为相比较前者更稳定

· 安装git

Git是分布式版本控制系统，GitHub可以托管各种Git版本库，并提供一个web界面。其目的是为了将文件上传到自己的github上

· 申请github账号

搭建博客

使用npm工具安装hexo

$ sudo npm install -g hexo

测试hexo是否安装成功(可选)

$ hexo

初始化hexo文件夹

$ hexo init

git

$ npm install hexo-deployer-git --save

生成静态页面

$ hexo generate（hexo g 也可以）

启动本地服务

$ hexo server (hexo s 也可以)

此时终端会出现http://localhost:4000/，用浏览器将其打开，一个最初的用hexo搭建的个人博客就呈现在你的面前了哦！！！
